About

I am a marketer turned software engineer with an interest in building tools to automate daily tasks. I am always looking for more opportunities to learn new skills, programming languages and frameworks. I am currently working on a personal recipe project using React Hooks and PostgreSQL with REST-ful API backend routes.

I recently built a web app called Watchwood where users can report incidents in San Francisco, using React and Node.js/Express.js. Also, during my bootcamp at Westcliff, I worked with a few teammates to build the website of EVFY - an electric vehicle charging solution company. I also created a full-stack single page web application called Kineto (a Netflix clone). I leveraged a database using PostgreSQL with REST-ful API backend routes as well as React, Redux, and CSS.
